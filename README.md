# Simple_Rust_Muti_Thread_Web_Server
A simple multi-thread web server in rust.

# prerequisite
```
rustc 1.30.1 (1433507eb 2018-11-07)
```

# Install
```
# git clone https://github.com/SamuelXing/Simple_Rust_Muti_Thread_Web_Server.git
```

# Build
```
cargo build
```

# Run
```
cargo run
```
